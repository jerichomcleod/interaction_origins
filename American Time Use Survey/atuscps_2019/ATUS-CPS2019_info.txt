
The data file atuscps_2019.dat (2019 ATUS-CPS file) contains 382 variables and 64,285 observations.  

For more information about this file, see the 2019 ATUS-CPS Data Dictionary, available online at:  https://www.bls.gov/tus/atuscpscodebk19.pdf.

In addition to the data file, three computer programs are contained in atuscps-2019.zip.  If you have SPSS, SAS, or Stata software, you may use these programs to read the data into your statistical software package.  The programs include variable and value labels for most variables.  For information about unlabeled values, please refer to the data dictionary.

The file "atuscps_2019.sps" reads the data into SPSS; some variable names exceed length 8.

The file "atuscps_2019.sas" reads the data into SAS.

The file "atuscps_2019.do" reads the data into Stata.

If you have SAS, SPSS, or Stata software, open the corresponding program, edit the infile (in SAS), file (in SPSS), or import (in Stata) statement to reference the data file on your computer, and run the program.
